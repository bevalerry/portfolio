# portfolio
Учебные проекты по программной инженерии: курсовые работы, лабораторные и практические задания по проектированию, разработке и тестированию программных систем.

4 семестр — Веб-приложение ресторана (Spring, MVC, REST)

Разработка серверной части и архитектуры веб-приложения для ресторана.
Реализовано MVC, REST API, подключение к БД, функциональность для отображения данных и взаимодействия с клиентом.
Ссылка на проект: https://github.com/bevalerry/restore

5 семестр — Backend сервиса аренды автомобилей (Spring, Java, REST)

Проектирование архитектуры и создание серверной части сервиса аренды авто.
Определены основные сущности, реализован REST API и структура приложения.
Ссылка на проект: https://github.com/bevalerry/car-rental-service

6 семестр — MERN-приложение для сбора отзывов

Разработка полноценного клиент-серверного приложения на MERN-стеке.
Реализованы backend, frontend, база данных, подключение и деплой в облако.
Ссылка на проект: https://github.com/bevalerry/rcsp_back
                  https://github.com/bevalerry/rcsp_front

7 семестр — Информационная система «Автосалон» (Проектирование, UML, БД)

Проектирование клиент-серверной системы для автоматизации процессов автосалона.
Созданы диаграммы, архитектура системы и логическая модель базы данных.
Ссылка на проект: https://github.com/bevalerry/coursework-7
